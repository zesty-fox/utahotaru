## 💜 简介 / About

> **StrangeUtaGame**（简称 **SUG**）是一款受 [RhythmicaLyrics](http://suwa.pupu.jp/RhythmicaLyrics.html) 启发的**卡拉 OK 歌词打轴软件**。它在保留 RhythmicaLyrics工作流的同时，重写了音频引擎与渲染管线，专注于**低延迟、高精度**的时间标签制作，并把对中文创作者友好的体验放在首位 —— **无需日语转区**、**全格式音视频导入**、**高质量变速**、**演唱者管理**。  
>
> **StrangeUtaGame (SUG)** is a karaoke timing tool inspired by *RhythmicaLyrics*. It rebuilds the audio engine and rendering pipeline around **ultra-low latency and frame-accurate timestamping**, with a focus on a smoother experience for Chinese-speaking creators — **no Japanese locale switch required**, **import any audio/video format**, and **studio-grade time-stretching**, **Singer Management**.

## 📦 安装与运行 / Installation

> 🪟 **支持系统 / Supported OS：** Windows 10 / 11

### 🚀 方式一 · 直接运行（推荐） / Pre-built release (recommended)

从 [Releases](https://github.com/Xuan-cc/StrangeUtaGame/releases) 页面下载最新版本，解压后双击 `StrangeUtaGame.exe` 即可。  
Download the latest archive from the [Releases](https://github.com/Xuan-cc/StrangeUtaGame/releases) page, unzip, then run `StrangeUtaGame.exe`.

<sub>📦 体积参考 / Approx. size: <b>~250 MB</b></sub>

---

### 🛠️ 方式二 · 从源码运行 / Run from source

```bash
# 克隆仓库 / Clone
git clone https://github.com/Xuan-cc/StrangeUtaGame.git
cd StrangeUtaGame

# 创建虚拟环境（推荐） / Create a virtualenv (recommended)
python -m venv venv
venv\Scripts\activate

# 安装依赖 / Install dependencies
pip install -r requirements.txt

# 启动 / Launch
python main.py
```

---

### 📦 方式三 · 自行打包 / Build it yourself

```bash
# 安装 PyInstaller / Install PyInstaller
pip install pyinstaller

# 使用打包脚本（推荐） / Run the build script
python build.py
```

打包完成后可在 `dist/StrangeUtaGame/` 目录找到可执行文件。  
The packaged executable will appear in `dist/StrangeUtaGame/`.

## ❓ FAQ

<details>
<summary><b>Q1. 为什么叫 "StrangeUtaGame"？这是音游吗？</b> &nbsp; <i>Why "StrangeUtaGame"? Is it a rhythm game?</i></summary>

<br/>

不是 🙅 —— 这只是个有点意思的名字。**Uta** 就是日语「歌」，项目本质是**卡拉 OK 打轴工具**，不是音游。  
No 🙅 — the name is just a bit of fun. **Uta** means "song" in Japanese; the project is a **karaoke timing tool**, not a rhythm game.

</details>

<details>
<summary><b>Q2. 和 RhythmicaLyrics、Aegisub-kara-templater 有什么区别？</b> &nbsp; <i>How is it different from RhythmicaLyrics or Aegisub kara-templater?</i></summary>

<br/>

- 相比 **RL**：现代 UI、毫秒级低延迟回放、无需日语转区、原生 UTF-8。  
- 相比 **Aegisub kara-templater**：SUG 专注「**打轴**」这一步，导出后再交给 Aegisub 做样式与特效，两者互补。  

</details>

<details>
<summary><b>Q3. 支持 macOS / Linux 吗？</b> &nbsp; <i>Does it support macOS / Linux?</i></summary>

<br/>

目前仅官方支持 **Windows 10/11**。源码本身是 Python + Qt，理论上跨平台 —— 欢迎社区贡献构建脚本与测试报告。  
Officially **Windows 10/11** only for now. The code is Python + Qt and should be portable; build scripts & test reports for other OSes are very welcome.

</details>

<details>
<summary><b>Q4. 我之前的 RhythmicaLyrics工程能直接导入吗？</b> &nbsp; <i>Can I import my existing RhythmicaLyricsproject?</i></summary>

<br/>

支持导入 RhythmicaLyrics导出的 ASS / LRC / TXT 与 <b>RL 编辑模式</b> 文本格式；联网词典同时兼容 RhythmicaLyrics 官方接口，原有词典资产可直接复用。  
ASS / LRC / TXT and the <b>RL edit-mode</b> text format from RhythmicaLyricscan be imported; the online dictionary is wire-compatible with the RhythmicaLyrics official endpoint, so your existing dictionary assets keep working.

</details>

<details>
<summary><b>Q5. 打开后没有声音 或 加载某个音频报错？</b> &nbsp; <i>No sound, or an audio format fails to load?</i></summary>

<br/>

发行版自带全套 <b>BASS DLL</b>（含 FLAC / OPUS / AAC / AC3 / WMA / DSD / ALAC / WebM 插件）位于 <code>lib/</code>；若从源码运行请保证 <code>src/strange_uta_game/lib/</code> 下的插件完整。视频文件由内置 <code>video_converter</code> 抽轨。  
The release ships the full <b>BASS</b> codec set (FLAC / OPUS / AAC / AC3 / WMA / DSD / ALAC / WebM) under <code>lib/</code>. From source, keep the DLL set at <code>src/strange_uta_game/lib/</code> intact.

</details>

<details>
<summary><b>Q6. 自动注音怎么工作？为什么某些词的读音不准？</b> &nbsp; <i>How does auto-furigana work? Why are some readings off?</i></summary>

<br/>

主引擎是 Windows 系统自带的 <b>WinRT JapanesePhoneticAnalyzer</b>。首次运行需在 设置 → 时间和语言 → 语言 中启用 <b>日语基本包 (Language.Basic~~~ja-JP)</b>；未启用时应用内会探测并引导安装。本地 / 联网词典会覆盖 IME 结果，遇到生僻词可补充本地词典。  
The primary engine is Windows' <b>WinRT JapanesePhoneticAnalyzer</b>, which needs the <b>Japanese basic IME pack</b>. The app probes for it and guides you to install. The local + online dictionaries override IME results for tricky readings.

</details>

<details>
<summary><b>Q7. 联网词典不走？代理 / 镜像怎么设？</b> &nbsp; <i>Online dictionary blocked? Proxy / mirror?</i></summary>

<br/>

在 <b>设置 → 网络</b>可切换词典源（内置 <b>RhythmicaLyrics 官方</b>）、自定义 URL、调整源优先级。自动更新则内置 <b>ghproxy / gh-proxy / ghproxy-net</b> 三个 GitHub 镜像。  
Under <b>Settings → Network</b> you can switch dictionary sources (<b>RhythmicaLyrics official</b>), add custom URLs, reorder priorities. The updater can route through <b>ghproxy / gh-proxy / ghproxy-net</b>.

</details>
