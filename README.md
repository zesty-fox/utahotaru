# StrangeUtaGame

一款由 RhythmicaLyrics 启发的歌词打轴软件，专注于低延迟、高精度的卡拉OK时间标签制作。

**当前版本**：v0.3.0 | **许可证**：GPL-3.0 | **平台**：Windows 10/11

## 功能概览

### 🎯 核心功能
- **精准打轴** — 类似节奏游戏的打轴体验，双模式快捷键系统（打轴模式/编辑模式）
- **变速播放** — 50%~200% 速度调节，pedalboard WSOLA 离线预渲染 + SPSC RingBuffer 零爆音
- **日语注音** — 基于 SudachiPy 的上下文感知自动注音，内置 1757 条常用词典
- **英文注音** — 基于 CMU 词典 + morikatron 规则引擎的英文假名注音

### 📝 编辑功能
- **多演唱者** — 逐字级别演唱者指定，颜色区分，拖拽划词选择
- **卡拉OK预览** — 实时走字高亮，Ruby 注音同步，连词平滑渲染
- **全文本编辑** — 自由修改歌词文本，智能保留时间标签和注音数据
- **行编辑** — 表格视图，支持行/字符级增删改操作
- **批量编辑** — 批量替换文本、注音和节奏点

### 🎵 音频处理
- **低延迟播放** — sounddevice + soundfile 多格式支持
- **实时变速** — 速度切换时实时显示渲染进度
- **Offset 校准** — 可视化节拍器校准窗口，精确补偿延迟

### 📁 文件支持
- **多格式导入** — LRC、ASS、SRT、TXT、KRA、Nicokara
- **多格式导出** — LRC（增强型/逐行/逐字）、KRA、SRT、TXT、txt2ass、ASS、Nicokara
- **项目文件** — .sug JSON 格式，支持版本迁移

### ⚙️ 系统功能
- **定时自动保存** — 可配置周期保存，支持闪退恢复
- **拖拽加载** — 音频/歌词/项目文件直接拖入窗口
- **深色/浅色主题** — 支持浅色、深色、跟随系统三种主题模式
- **快捷键自定义** — 支持组合键和双快捷键绑定

## 快速开始

### 方式一：直接运行（推荐）

从 [Releases](https://github.com/Xuan-cc/StrangeUtaGame/releases) 页面下载最新版本的压缩包，解压后运行 `StrangeUtaGame.exe` 即可。

**体积参考**：约 410 MB（含日语形态素分析词典）

### 方式二：从源码运行

```bash
# 克隆仓库
git clone https://github.com/Xuan-cc/StrangeUtaGame.git
cd StrangeUtaGame

# 创建虚拟环境（推荐）
python -m venv venv
venv\Scripts\activate

# 安装依赖
pip install -r requirements.txt

# 启动应用
python main.py
```

### 方式三：自行打包

```bash
# 安装 PyInstaller
pip install pyinstaller

# 使用打包脚本（推荐）
python build.py
```

打包完成后可在 `dist/StrangeUtaGame/` 目录找到可执行文件。

## 使用流程

### 1. 加载文件

启动后默认进入打轴界面，直接拖入文件即可开始：

- **拖入歌词文件**（LRC / ASS / SRT / TXT / KRA / Nicokara）→ 加载歌词
- **拖入音频文件**（MP3 / WAV / FLAC / OGG）→ 加载音频
- **拖入项目文件**（.sug）→ 打开已有项目

也可以通过侧边栏导航到「主页」手动创建项目。

### 2. 打轴

编辑器有**打轴模式**和**编辑模式**两种模式，左下角实时提示当前模式。音乐播放时自动进入打轴模式，暂停时进入编辑模式。

**上方工具栏**：
- **文件管理**：新建/加载/保存项目，加载音频/歌词
- **修改所选字符**：批量替换文本、注音和节奏点
- **插入导唱符**：在选中字符前批量插入导唱用占位符
- **批量变更**：批量替换注音、删除注音、设置节奏点
- **修改选中行**：编辑当前行的属性
- **注音分析**：为选中字符生成自动注音
- **按类型删除注音**：选择性移除特定类型的注音
- **按行设置演唱者**：批量设置当前行的演唱者
- **全局偏移**：调整渲染/导出的时间偏移量（默认 -230ms，范围 -5000~5000ms）

**播控栏**：
- **播放/暂停/停止**：控制音频播放
- **进度条**：拖动跳转到指定位置
- **速度控制**：50%~200% 变速播放（默认 100%），支持直接输入百分比
- **音量控制**：调整播放音量（默认 80%）
- **渲染进度**：变速时显示后台 WSOLA 渲染进度

| 按键 | 打轴模式 | 编辑模式 |
|------|---------|---------|
| `Space` | 打轴 | 增加节奏点 |
| `D` | 播放 / 暂停 | 播放 / 暂停 |

| `Q` / `W` | 减速 / 加速 (±10%) | 减速 / 加速 (±10%) |
| `↑` / `↓` | 上一行 / 下一行 | 上一行 / 下一行 |
| `←` / `→` | 上一字符 / 下一字符 | 上一字符 / 下一字符 |
| `Enter` | — | 插入换行 |
| `Delete` | — | 删除选中字符 |
| `Backspace` | — | 减少节奏点 |
| `F2` | 编辑注音 | 编辑注音 |
| `F3` | 连词 / 取消连词 | 连词 / 取消连词 |
| `F5` / `F6` | 增加 / 减少节奏点 | — |
| `.` (长按) | 切换句尾 | — |

| `Alt+↑` / `Alt+↓` | 当前时间戳 ± 10ms | 当前时间戳 ± 10ms |
| `Ctrl+Z` / `Ctrl+Y` | 撤销 / 重做 | 撤销 / 重做 |
| `Ctrl+S` | 保存项目 | 保存项目 |

> 快捷键可在「设置 → 快捷键」中自定义，支持组合键、双快捷键绑定和长按绑定（按键持续 300ms 以上触发）。

### 右键菜单

在打轴预览中右键点击字符，可快速执行以下操作：

- **删除字符**：删除选中字符（支持划词多选）
- **删除当前时间戳并回滚**：移除当前节奏点的时间戳
- **在此插入空格**：在当前字符后插入空格
- **合并上一行**：将当前行合并到上一行末尾
- **删除本行**：删除当前整行
- **在此插入空行**：在当前行前插入空行
- **增加/减少节奏点**：调整当前字符的节奏点数量
- **设置/取消句尾**：切换句尾标记
- **设置演唱者**：为选中字符指定演唱者（支持划词多选）

### 3. 注音与编辑

- **自动注音**：在全文本编辑界面点击「自动分析全部注音」，支持汉字、假名、英文、数字等全字符类型
- **手动调整**：F2 键编辑单字注音，全文本编辑支持 `{原文||读音组}` 连词格式
- **按类型删除**：可选择性移除平假名、片假名、英文等不需要的注音

### 4. 保存与导出

- `Ctrl+S` 保存项目（.sug 格式）
- 侧边栏「导出」→ 选择格式 → 设置文件名 → 导出

**支持导出格式**：LRC（增强型/逐行/逐字）、KRA、SRT、TXT、txt2ass、ASS、Nicokara

### 5. 演唱者管理

- 在打轴预览中拖拽选中文字 → 右键 → 选择演唱者
- 支持逐字级别的演唱者指定
- 演唱者预设跨项目保持

## 项目文件

| 格式 | 说明 |
|------|------|
| `.sug` | StrangeUtaGame 项目文件（JSON 格式，v0.3.0） |
| `.sug.temp` | 自动保存临时文件（存放在程序目录 `.cache` 文件夹下，正常退出时自动清理） |

**特性**：
- 项目文件不保存音频路径，每次使用时需重新选择音频文件
- 支持 v0.1 → v0.3.0 自动迁移（Ruby → RubyPart 结构升级）
- 配置文件分离：`config.json`（主配置）、`dictionary.json`（用户词典）、`singers.json`（演唱者预设）

## 常见问题

**Q: 打轴时延迟很大怎么办？**
在设置 → Offset 校准中使用可视化校准工具调整偏移量。

**Q: 变速后声音有杂音？**
变速采用 pedalboard WSOLA 离线预渲染 + SPSC RingBuffer 架构，正常情况下不会有爆音。如遇异常请截取片段反馈。

**Q: 如何使用已有的 RhythmicaLyrics 字典？**
在设置 → 用户字典中点击「导入 RL 字典」。

**Q: 项目打不开？**
确认文件扩展名为 `.sug`，且为有效的 JSON 格式。启动时如检测到自动保存文件会提示恢复。

**Q: 如何自定义快捷键？**
在设置 → 快捷键中配置，支持组合键和双快捷键绑定，冲突即时检测。

**Q: 如何切换主题？**
在设置 → 主题中选择浅色、深色或跟随系统模式。
