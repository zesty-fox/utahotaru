🎚️ 偏移与校准

软件里有好几个「偏移」，含义各不相同。这一章讲清每个偏移、如何校准、如何检查轴准不准。

先理解：为什么会有偏移？

你「听到声音 → 大脑反应 → 手指按键」本身就有约 0.2 秒的固有延迟；耳机/声卡/系统输出也各有延迟。 如果直接把按键时刻当成字的开始时间，整条轴会系统性偏晚。偏移设置就是用来抵消这些延迟、让轴对得准。

🧩 三种偏移，一次看懂

偏移默认作用时机解决什么
按键补偿
tag_offset_ms		 0 ms 打轴那一刻，每次按键即时生效 抵消「你的反应 + 设备」延迟，让记录的时间戳比你实际按键提前一点
全局偏移
软件内 / 编辑器		 0 ms 打完轴后，在软件内实时统一平移全部时间戳（预览即时刷新） 轴整体早/晚了一截，不重打、一键整体挪；按项目保存
导出偏移
export.offset_ms		 0 ms 作用于导出文件的整体偏移 适配某个播放器/平台时，给导出结果再整体挪一挪
⚠️ 速度补正不是偏移

「速度补正」（speed_correction，默认 100%，范围 50–200%）是修正系数，不在这三种偏移之列。 它在变速打轴（如 0.6× 慢放）时，对记录的时间戳做速度相关修正，让慢放打的轴换回原速也准。一般保持默认即可。

💡 三者怎么分工

按键补偿是打轴时边打边补，影响「新打的字」；全局偏移是打完后把 「所有已存在的字」在软件内统一搬家；导出偏移则作用在最终导出的文件上。 三者都遵循负值 = 提前、正值 = 延后。

① 按键补偿（tag_offset_ms）

位置：设置 → 打轴 → 按键补偿。默认 0 ms，范围 −5000～5000，步进 10 ms。

  • 负值表示「把时间戳记得比按键时刻更早」—— 人通常慢半拍按下，所以校准后这个值往往是负的。
  • 不建议凭感觉乱填，强烈建议用下面的节拍器校准测出你自己的值再填进去。

② 全局偏移

位置：打轴界面工具栏的「全局偏移」输入框（范围 −5000～5000 ms）。

  • 把整首歌所有时间戳在软件内统一平移，预览即时刷新，所见即所得。
  • 按项目保存进 .sug 工程，下次打开自动恢复。
  • 它与导出偏移联动同步：改了全局偏移，导出偏移会跟着更新，所以预览看到的就是导出得到的。

③ 导出偏移（export.offset_ms）

作用于导出文件的整体偏移，默认 0 ms。它与上面的全局偏移是同一套整体偏移的两个作用面、 保持同步 —— 一般你只需调全局偏移即可，导出会自动带上同样的偏移；只有当你想让导出结果与软件内预览 刻意错开（比如专门适配某播放器）时才单独关注它。

别和「微调时间戳步长」搞混

设置 → 打轴 → 微调时间戳步长（默认 10 ms）决定打轴界面里 Alt+↑ / Alt+↓ 每次微调单个节奏点的幅度 —— 这是针对一个字的精修，不是整体偏移。

🎼 用节拍器校准测出你的「按键补偿」

位置：设置 → 打轴 → Offset 校准 → 开始校准。它能客观量出「你听到拍子后平均晚多少按下」。

  1. 打开校准弹窗深色画布上有滑块匀速滑过中央的判定线，同时播放节拍器「咔哒」声。
  2. 戴好耳机，跟着节拍按空格每当滑块/咔哒到达判定线，就按一下 Space。
  3. 看实时统计左上角显示你最近一次与平均偏移；可调 BPM（60–240）多敲几轮取稳。
  4. 把平均值填进「按键补偿」用校准得到的偏移设置 tag_offset_ms， 之后打轴就自动抵消你的固有延迟。
⚠️ 换设备要重校

偏移和你的耳机/声卡/蓝牙都有关。换了音频设备（尤其蓝牙耳机延迟很大）建议重新校准。 用蓝牙打轴往往延迟大且不稳，追求精度时优先有线设备。

🔍 如何检查轴精度

打完轴别急着导出，用这几种方式自查准不准：

1. 双击字 / 节奏点，跳到该处听声

这是最直接的「点对点」核对法：双击某个字，音频会立刻跳到这个字时间戳的位置开始播放； 双击某个节奏点，则跳到该节奏点的时间。于是你能逐字/逐点听原声，确认轴有没有对在咬字起点上。

  • 是双击才会跳音频听声。单击不会 —— 单击只是把光标/选中移到该字或节奏点 （决定下次从哪打、哪里开始播），并不会跳到该处发声。
  • 双击跳转不受「设置 → 打轴 → 禁用单击跳转」影响，随时可用。

2. 看卡拉 OK 预览逐字变色

播放时盯着预览区，文字应恰好在演唱者唱到该字时变色。 早了说明轴偏前，晚了说明偏后 —— 据此调整。

3. 打开「打轴预览指引」

设置 → 打轴 → 打轴预览指引：播放时用过渡色提示「上一个 / 正在打 / 下一个」字， 以光标为锚，让你更直观看出当前打到哪、对没对上。各档透明度与开关可在「预览指引方式」里调。

4. 用波形 / 时间线核对

波形图上的能量起伏对应发音起止。把节奏点位置和波形的音头对一对，能看出明显的错位。

5. 慢放复听

按 Q 减速到 0.2×–0.6×，逐字听咬字起点，配合 Z 回放反复确认。

6. 逐点微调

发现某个字偏了，选中它用 Alt+↑ / Alt+↓ 按步长（默认 10 ms）微调；整体偏就改全局偏移。

💡 排查顺序建议

① 想确认某个具体字 → 双击它跳转听声；② 整条轴都偏一致 → 调全局偏移或重测按键补偿； ③ 只有个别字偏 → 逐点微调；④ 慢放打的轴换原速整体偏 → 检查速度补正。
