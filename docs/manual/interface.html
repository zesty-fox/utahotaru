🪟 主窗口与侧边栏

SUG 采用 Windows 11 风格的 Fluent 界面，左侧是导航侧边栏，点击切换不同的工作界面：

导航项用途
🏠 主页创建项目入口：拖入 / 选择音频与歌词，点「创建项目」。
🎯 打轴核心工作区：波形、卡拉 OK 预览、歌词字符行、打轴操作全在这里。
🎤 演唱者管理演唱者列表：颜色、分色、分组、启用/禁用、预设。
🌐 在线词典联网读音词典查询界面。
📤 导出选择格式与选项，导出歌词 / 字幕文件。
⚙️ 设置10 个分类的全局配置页。
全局快捷操作

Ctrl+S 随时保存工程（无路径时弹出另存为）。 启动时若检测到上次异常退出，会询问是否恢复闪退现场。关闭时若有未保存改动，会弹出 「保存 / 放弃 / 取消」确认。

🎯 打轴界面剖析

这是你花最多时间的地方。它由上到下大致分为几块：

① 顶部传输条（Transport Bar）

播放控制中枢，包含：

  • 播放 / 暂停 / 停止按钮（对应 D / S）。
  • 变速：可直接在输入框填 20–200%，或用 Q/W 每次 ±5%（范围 0.2×–2.0×，变速不变调）。
  • 音量：Ctrl+Q / Ctrl+W 每次 ±5%。

另外，编辑器工具栏上有一个「全局偏移」输入框（范围 −5000～5000 ms）， 对整首歌所有时间戳统一平移、预览即时刷新，按项目保存，并与导出偏移联动。 按键补偿 / 全局偏移 / 导出偏移三者的区别见 偏移与校准。

② 卡拉 OK 预览（KaraokePreview）

实时模拟卡拉 OK 走字效果：播放时逐字高亮、注音同步走字，让你边打边看效果对不对。

  • 无时间戳的字会按邻近组时间均分「连读」显示。
  • 连词组（多个字共享一段读音）会合并渲染。
  • 可叠加「上一个 / 正在打 / 下一个」字的引导着色，每档透明度与开关可在设置里调。
  • 没打时间戳的位置可标「此处需导唱符」待办标记，提醒后续补打。

③ 波形 / 时间线（TimelineWidget）

显示当前音频波形与节奏点位置，帮你用眼睛辅助判断字的起止 —— 配合慢放扒轴尤其有用。

④ 歌词字符行

逐行逐字显示歌词，每个字上方是它的注音，下方/内部承载它的节奏点（checkpoint）与时间戳。 这里是打轴的「画布」：

  • 单击某字：跳到它的第一个节奏点；无节奏点的字会保持视觉焦点便于添加。
  • 划选多字：跨行框选，用于批量操作（设演唱者、删时间戳等）。
  • 右键某字：弹出上下文菜单 —— 删除字 / 删时间戳 / 插入空格 / 合并行 / 删除行 / 插入空行 / 增减节奏点 / 切换句尾 / 设置演唱者。

⑤ 工具栏（ToolBar）

各种批量操作的指挥中心，分成「文件管理 / 编辑管理 / 插入导唱符 / 自动注音管理 / 演唱者相关 / 全文本编辑 / 时间戳工具」几组下拉菜单与按钮，外加一个「全局偏移」输入框。 每项的详细用法见 打轴详解 → 工具栏全览。

🔀 两种工作模式：打轴模式 vs 编辑模式

打轴界面有两种模式，仅 Space 与 Backspace 的行为不同：

模式SpaceBackspace适用
打轴模式 在当前字打时间戳 清除当前字时间戳 实际听音、记录时间
编辑模式 增加节奏点 减少节奏点 调整每个字的节奏点数量 / 拆分细节
什么是「节奏点 / checkpoint」？

一个字可以有多个节奏点（比如长音、拖腔需要中途换色）。节奏点决定这个字在卡拉 OK 里 分几段变色。大多数字只有 1 个；编辑模式就是用来给特殊字增减节奏点的。

📝 其它编辑视图

行编辑 / 行详情

以表格形式总览所有歌词行（行号、演唱者、连词标记等），支持加/删/复制/插入行；双击行打开 行详情对话框，可逐字编辑注音、节奏点、句尾标记和演唱者。

全文本编辑

把整段歌词作为一段带注音与时间戳的内联文本来编辑 —— 注音 {原文||读音}、时间戳 [mm:ss.cc]、句尾、演唱者 【名】 全写在文本里， 键盘流批量改字、改注音、调时间一气呵成。Ctrl+T 打开。 它自带完整时间轴、逐行独立解码，行的增删/重排不会错配时间戳。详见 全文本编辑详解。
