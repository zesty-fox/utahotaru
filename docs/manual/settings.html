⚙️ 设置详解

设置界面顶部用选项卡（Pivot）分成 10 个分类页。本章逐页讲解。

通用操作

设置改动会防抖自动保存（约 500 ms 后写盘）。底部还有「保存设置」「从 KS 配置导入」 「重置为默认设置」按钮。重置不影响用户词典与演唱者预设（它们独立存储）。

1️⃣ 演奏控制

音频回放相关默认值：

项说明
默认音量音频加载后的初始音量
默认速度音频加载后的初始播放速度
速度滑块最小值 / 最大值播放栏速度滑块的上下限（默认 0.2×–2.0×）
快进量 / 后退量按快进/后退键跳过的毫秒数（默认 3000 / 2000 ms）
加载后自动播放导入音频后是否立即开始播放
高质量变速开启更高质量的时间拉伸算法
删除回跳量清除节奏点后自动回跳的毫秒数（便于复听）
滚动模式歌词滚动：自动 / 始终 / 从不

2️⃣ 打轴

打轴行为与时间补偿，含可视化 Offset 校准工具：

项说明
按键补偿（tag_offset_ms）偏移①：每次打轴的延迟补偿，默认 0 ms（负=提前，正=延后）；建议用下方校准测出你的值再填
速度补正变速打轴时的时间戳修正系数（不是偏移），默认 100%（50–200%）
全局偏移 / 导出偏移（export.offset_ms）偏移②③：整首歌时间戳的整体平移，默认 0、±5000 ms；与打轴界面工具栏的「全局偏移」联动同步，导出时一并带上
微调时间戳步长Alt+↑/↓ 每次微调的毫秒（默认 10 ms）
禁用单击跳转关掉「单击字符/节奏点延迟后跳转到目标行」（双击跳转不受影响）
隐藏焦点高亮隐藏 current / focus 域的高亮底色（启用后仅拖拽多选时显示）
打轴预览指引预览叠加「上一个/正在打/下一个」引导着色，各档透明度与开关可调
打轴按键音 / 音量 / 风格按下/抬起音效开关（默认关）、音量（0–200%）、风格（默认 / osu / 街机风 / 金属感）

三种偏移的详细含义、如何校准、如何检查轴精度，见 偏移与校准。

🎚️ Offset 校准弹窗

深色画布上两个白滑块按设定 BPM 匀速滑过中央判定线，同时播放节拍器点击音 —— 跟着节拍按空格，软件实时算出你的平均按键偏移，把它填进「按键补偿」就能抵消 系统/耳机延迟。BPM 60–240 可调。

3️⃣ 自动保存

项说明
启用定时自动保存开/关周期性自动落盘
自动保存间隔1–60 分钟（默认 5）

自动保存写入程序目录 .cache 下的临时文件；异常退出后启动会提示恢复。

4️⃣ AutoCheck（节奏点自动规则）

控制「自动补节奏点」按字符类型的启停 —— 决定哪些字会被自动打上节奏点。可逐类开关：

ひらがな 平假名カタカナ 片假名 漢字アルファベット 英文字母 数字記号 符号空格

另有 20+ 项细则：小写假名、句首/句尾判定、把空格当行尾、标点是否打点、括号、英文音节、 促音、长音「ん」等。AutoCheck 的「节奏点」流程与「注音」流程相互独立。

5️⃣ 读音词典

管理本地用户词典与网络词典源：

  • 自定义读音：增删条目，上移 / 下移调整优先级（新增置顶）。
  • 导入 RL 字典：原文⇥读音1,读音2…，＋ 表连词。
  • 启用网络词典与字典源优先级：本地源 + 各网络源自顶向下排序，首个命中即停。

详见 注音详解 → 本地用户词典 / 联网读音词典。

6️⃣ 界面

项说明
主题自动（跟随系统）/ 浅色 / 深色
主文字字体 / 注音字体分别指定歌词正文与注音的字体
基础字体大小非当前行歌词的像素大小
当前行字体大小当前行歌词的像素大小（通常更大）
注音字体大小Ruby 注音的像素大小
注音与主文字间距注音与正文之间的像素间距

7️⃣ 导出

设置默认导出格式及各格式的默认选项 —— 11 种格式：LRC（增强型/逐行/逐字）、KRA、TXT、 SRT、txt2ass、ASS、Nicokara、Nicokara（带注音）、RL 编辑模式。详见 导出格式详解。

8️⃣ 快捷键

重新映射所有打轴/编辑快捷键。支持：

  • 双绑定：一个功能绑两个键。
  • 长按绑定：同一个键短按与长按（持续 300 ms 以上）触发不同功能。

完整默认键位见 快捷键大全。

9️⃣ 网络

分两块：

  • 词典源：内置 RhythmicaLyrics 官方源，可自定义源 URL、调整源顺序、定时自动更新。
  • 更新与代理：自动更新可走 ghproxy / gh-proxy / ghproxy-net 三个 GitHub 镜像， 支持系统代理 / 手动代理 / 跳过某版本。
