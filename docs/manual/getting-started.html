🚀 新手入门

从下载到导出第一份卡拉 OK 时间轴，跟着走一遍就懂了。

🎯 先理解「打轴」这件事

「打轴」= 给歌词的每一个字标上时间。SUG 的做法很像音游：你戴上耳机、放歌， 每听到一个字开始唱，就在那一刻按一下键。软件把你按键的时刻记成这个字的开始时间， 逐字记录下来，就形成了一条「时间轴」。导出后，卡拉 OK 播放器就能让歌词跟着音乐逐字变色。

① 安装与启动

方式一 · 直接运行（推荐新手）

  1. 下载发行版 前往 Releases 页面， 下载最新的 StrangeUtaGame-v1.2.6.zip（约 250 MB）。
  2. 解压到任意目录 建议放在一个英文或中文路径都可以的固定文件夹（SUG 原生支持 UTF-8 路径，无需日语转区）。
  3. 双击运行 打开文件夹里的 StrangeUtaGame.exe 即可。首次启动会显示启动画面（Splash）。
🍎 macOS 用户

macOS 版为实验性构建，未签名。首次打开请在 StrangeUtaGame.app 上 右键 → 打开，弹窗里再点一次「打开」；或在终端执行 xattr -dr com.apple.quarantine /路径/StrangeUtaGame.app。 macOS 版不支持应用内自动更新，新版本会用浏览器打开 Releases 页面供手动下载。

方式二 · 从源码运行（开发者 / 进阶）

git clone https://github.com/karaoke-studio/StrangeUtaGame.git
cd StrangeUtaGame
python -m venv venv
venv\Scripts\activate
pip install -r requirements.txt
# Windows 主版本（WinRT 注音）额外装：
pip install -r requirements-winrt.txt
python main.py

noWinIME / mac 变体改装 requirements-variants.txt（使用 Sudachi 注音）。详见仓库 README 的「安装与运行」。

② 准备日语注音引擎（建议）

如果你打的是日文歌，希望自动给汉字标假名（注音 / furigana），主版本依赖 Windows 自带的 WinRT 日语语音分析器。它需要系统安装日语基本包：

  1. 打开系统设置Windows 设置 → 时间和语言 → 语言和区域。
  2. 添加日语添加语言 → 选「日本語」，安装其中的「基本输入」语言包 （Language.Basic~~~ja-JP）。
  3. 回到 SUG未启用时，软件会自动探测并弹出引导提示，按提示安装即可。
不打日文歌？

那可以跳过这一步。noWinIME / mac 变体内置 Sudachi 注音引擎，无需系统语言包。

③ 创建你的第一个项目

SUG 的项目 = 一首歌的「音频 + 歌词 + 时间轴」打包。创建非常简单 —— 直接拖进去就行。

  1. 准备素材 一段音频（mp3 / wav / flac / ogg 等，视频文件也行，会自动抽音轨）和一份歌词文本 （纯文本 .txt，或已有的 .lrc / .ass / .srt / .kra / Nicokara）。
  2. 拖入主页 把音频和歌词文件拖到软件窗口里即可。也可在「主页」用按钮选择文件。
  3. 点击「创建项目」 SUG 会自动识别歌词格式（ASS / SRT / LRC / Nicokara / 内联 / 纯文本），完成解析。
  4. （可选）自动注音 日文歌创建时会自动为未注音的字生成假名，不会覆盖你导入文件里已有的注音。

💡 没有现成歌词文件？

直接把歌词文字粘进主页的文本框，一行一句，也能创建项目。打轴前可以随时在界面里改字、断行。

④ 开始打轴（核心流程）

项目创建后会进入打轴界面。最基础的打轴只用到三个键：

键作用
D播放 / 暂停
Space打轴 —— 在「当前字」上记录此刻的时间戳，并自动跳到下一个字
Backspace清除当前字的时间戳（只清时间，不删字）
  1. 把光标放到第一行第一个字用方向键 ↑↓←→ 移动。
  2. 按 D 播放音乐开始，仔细听。
  3. 每听到一个字开唱，就按一下 Space 光标会自动前进到下一个字，跟着唱跟着按即可。
  4. 按错了？按 Z 后退 2 秒重听，把光标移回去， Backspace 清掉再重打。
  5. 太快跟不上？按 Q 减速（最低 0.2× 变速不变调），慢慢扒。
💜 新手节奏建议

第一次别追求一遍到位。先用 0.6×～0.8× 慢放，一句一句来；打完一句用 Z 回放， 看预览区文字是否跟着音乐准确变色，不对就局部重打。打轴是「听 → 按 → 校」的循环。

⑤ 保存与导出

  1. 随手保存工程 Ctrl+S 保存为 .sug 工程文件，下次可继续编辑。 （软件默认每 5 分钟也会自动保存一次。）
  2. 打开导出界面 在侧边栏切到「导出」，选择目标格式（卡拉 OK 常用 Nicokara 或 ASS；歌词常用 LRC）。
  3. 导出 确认选项后导出文件。以后同样的格式与目录可用 Ctrl+E 一键快速导出。
🎉 你已经完成了一份时间轴！
